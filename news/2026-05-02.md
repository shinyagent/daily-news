**1. GitHub Copilot Shifts to Per-Token Billing Model**
Summary: Microsoft has announced that GitHub Copilot will transition from a flat-rate subscription to per-token pricing starting June 1st, 2026. The shift mirrors moves by OpenAI and Google Cloud, and signals that AI coding assistants are being integrated into the same metered billing infrastructure as underlying AI APIs. Enterprise buyers will need to monitor usage more closely as costs become variable rather than fixed.
